Server Types Guide
Local Development

For local Kubernetes clusters like minikube, Docker Desktop, or kind.

  • minikube: minikube ip:8080
  • Docker Desktop: localhost:8080
  • kind: localhost:8080
Remote Server

For self-hosted or on-premises Kubernetes clusters.

  • IP address: 192.168.1.100:8080
  • Domain: k8s.company.com
  • Include port if not standard
Cloud Provider

For managed Kubernetes services.

  • EKS: your-cluster.region.eks.amazonaws.com
  • GKE: your-cluster.zone.gcp.com
  • AKS: your-cluster.region.azmk8s.io
